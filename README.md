## Featured Projects

### [Vulkan Voxel Engine](https://github.com/Oliver-Wilde/VulkanProject)

A real-time voxel engine built in **C++** using **Vulkan**, developed as part of my final-year Computer Science project work.

The project focused on chunk-based terrain rendering, GPU resource management, multithreaded chunk rebuilding, memory usage, rebuild latency, and frame-time variance.

**Key areas:**

* C++ systems programming
* Vulkan graphics programming
* Chunk-based voxel terrain
* GPU resource management
* Multithreaded rebuild scheduling
* Performance profiling and benchmarking
* Technical research and evaluation

---

### [Iterated Prisoner’s Dilemma Strategy Simulator](https://github.com/Oliver-Wilde/Prisoners-Dilemma-Oliver-Wilde)

A C++ simulation framework for evaluating strategies in the **Iterated Prisoner’s Dilemma**.

The project explores repeated-game behaviour, strategy evaluation, adaptive decision-making, noisy payoff conditions, and comparative experimental analysis.

**Key areas:**

* C++ software design
* Algorithmic simulation
* Game theory
* Strategy evaluation
* Experimental analysis
* Data-driven comparison of outcomes
